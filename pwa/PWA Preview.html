Deployment checklist
Качи manifest.webmanifest в корена на домейна (напр. app.meet-bg.com/manifest.webmanifest).
Качи папка icons/ с всички PNG файлове до манифеста.
Постави HTML snippet в <head> на всяка страница.
Сайтът трябва да се сервира по HTTPS — задължително условие за PWA инсталация.
Регистрирай Service Worker за offline поддръжка — препоръчваме Workbox.
Провери с Chrome DevTools → Application → Manifest за грешки при зареждане.
